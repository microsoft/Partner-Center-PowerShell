# Get-PartnerRole

## SYNOPSIS
Gets the roles at the partner level.

## SYNTAX

```powershell
Get-PartnerRole [<CommonParameters>]
```

## DESCRIPTION
Gets a list of roles at the partner level.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-PartnerRole
```

Gets a list of roles at the partner level.
